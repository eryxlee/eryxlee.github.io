  2. MAC OS 上安装MySQL 5.7.10后Python连接问题

    2015-12-03
    By Eryx Lee

    通过brew安装了最新版本的MySQL，在python中import MySQLdb发现找不到libmysqlclient.18.dylib文件。而MySQL 5.7.10带的lib文件是libmysqlclient.20.dylib。经过测试后，发现直接讲文件改名链接过来也是可以用的。

    sudo ln -s /usr/local/opt/mysql/lib/libmysqlclient.20.dylib /usr/local/lib/libmysqlclient.18.dylib
